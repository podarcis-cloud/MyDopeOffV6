# i18n — comment ajouter du contenu traduit

L'architecture multilingue (FR / EN / ES / DE) est dans `lucide.js`. Le « cadre »
(navigation, accueil, libellés communs) est déjà traduit. Pour le contenu —
notamment les fiches substances — rien à changer dans l'architecture : il suffit
de remplir des traductions.

## 1. Traduire un texte d'interface
Ajouter une clé dans l'objet `DICT` de `lucide.js` :

```js
'ma.cle': { fr: 'Bonjour', en: 'Hello', es: 'Hola', de: 'Hallo' },
```

Puis dans le HTML :
```html
<span data-i18n="ma.cle">Bonjour</span>          <!-- texte -->
<input data-i18n-ph="common.search">             <!-- placeholder -->
<div data-i18n-html="home.intro">…</div>          <!-- HTML -->
```
Tout est traduit automatiquement au chargement et à chaque changement de langue.

## 2. Traduire dynamiquement en JS
```js
LUCIDE.t('nav.substances');                        // par clé
LUCIDE.t({ fr:'Oui', en:'Yes', es:'Sí', de:'Ja' }); // objet direct
```

## 3. Traduire les FICHES (plus tard)
Les fiches restent en français tant qu'on ne fournit pas de traduction.
Le format prévu : un objet par langue, indexé par le NOM de la substance.

Créer un fichier `fiches-i18n.js` chargé après `lucide.js` dans psychochecker.html :

```js
window.LUCIDE_FICHES_I18N = {
  en: {
    "Cannabis": {
      effets: ["Relaxation","Euphoria","Altered perception"],
      description: "…",
      tripReport: "…"
    },
    // … une entrée par fiche
  },
  es: { "Cannabis": { … } },
  de: { "Cannabis": { … } }
};
```

Dans le rendu d'une fiche (psychochecker.html), envelopper les champs traduisibles :
```js
function tr(nom, champ, valeurFR){
  const l = LUCIDE.lang();
  const t = (window.LUCIDE_FICHES_I18N||{})[l];
  return (t && t[nom] && t[nom][champ]) ? t[nom][champ] : valeurFR;
}
// ex : tr(s.nom,'description', s.description)
```

Ainsi on traduit fiche par fiche, langue par langue, sans rien casser :
une fiche non encore traduite s'affiche simplement en français.

## Langues
`LUCIDE.LANGS` = ['fr','en','es','de']. Pour en ajouter une, l'ajouter à ce
tableau + `LANG_NAMES`, et compléter chaque entrée de `DICT`.
