# Lucide

![Status](https://img.shields.io/badge/status-active-success)

## À propos

Lucide est une application web progressive (PWA) conçue pour centraliser des informations de réduction des risques relatives aux substances psychoactives.

L'objectif du projet est de fournir un accès simple, rapide et structuré à des informations toxicologiques, pharmacologiques et préventives afin d'améliorer la compréhension des risques associés aux substances et à leurs interactions.

Lucide ne vise ni à promouvoir ni à encourager la consommation de substances. L'application s'inscrit dans une démarche de santé publique, d'information et de réduction des risques.

---

## Fonctionnalités

### Base de données substances

- Recherche instantanée
- Alias et noms alternatifs
- Classification pharmacologique
- Effets principaux
- Risques connus
- Informations toxicologiques
- Données de prévention

### Analyse d'interactions

- Analyse simultanée de plusieurs substances
- Détection des interactions documentées
- Mise en évidence des combinaisons à risque
- Alertes visuelles

### Application Web Progressive (PWA)

- Installation sur smartphone et ordinateur
- Fonctionnement hors ligne
- Interface optimisée mobile
- Chargement rapide

### Recherche avancée

- Recherche par nom
- Recherche par alias
- Filtrage par famille pharmacologique
- Navigation simplifiée

---

## Philosophie du projet

Lucide repose sur plusieurs principes :

- Information scientifique accessible
- Réduction des risques
- Neutralité et objectivité
- Transparence des données
- Accessibilité hors ligne
- Respect de la vie privée

Aucune donnée personnelle n'est collectée par l'application.

---

## Sources documentaires

Les informations intégrées au projet sont compilées à partir de ressources publiques reconnues dans les domaines de la pharmacologie, de la toxicologie et de la réduction des risques.

Les données sont systématiquement revues et structurées afin de privilégier la clarté, la cohérence et la prudence.

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/votre-utilisateur/lucide.git
```

Entrer dans le dossier :

```bash
cd lucide
```

Ouvrir :

```bash
index.html
```

Ou déployer via GitHub Pages.

---

## Utilisation

1. Ouvrir l'application.
2. Rechercher une substance.
3. Consulter la fiche correspondante.
4. Utiliser l'outil d'analyse pour comparer plusieurs substances.
5. Lire les alertes et recommandations affichées.
